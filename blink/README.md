# blink

The classic blink program, which blinks the on-board LED. Useful yo test your
setup.  The program uses some boilerplate generated by CubeMX, fortunately you
don't need to use it do develop and flash the program.


Build the binary:

```bash
make
```

Run using nt-link:

```bash
make flash-link
```

Run using UART:

```bash
make flash-uart
```
